# hubcap help

Show help for hubcap commands.

## When to use

Use `help` with no arguments to see all available commands grouped by category. Use `help <command>` to see detailed documentation for a specific command.

## Usage

```
hubcap help [command]
```

## Arguments

| Argument | Type | Required | Description |
|----------|------|----------|-------------|
| command | string | No | Command name to get help for |

## Examples

List all commands:

```
hubcap help
```

Get help for a specific command:

```
hubcap help goto
hubcap help assert
hubcap help screenshot
```
